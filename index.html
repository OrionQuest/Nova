Multi-Physics Simulation with Nova¶

Nova is a high-performance library for Computational Physics. In terms of design, it draws inspiration from its elder brother PhysBAM, but uses modern libraries such as STL, Boost, Assimp, etc., to reuse existing open-source implementations whenever possible. Parts of it have been (and continue to be) parallelized using OpenMP. Our goal is to make it a versatile library for various multi-physics problems and geometric computation. This documentation provides tutorials on how to use the library.
